# 🌍 COVID-19 Vaccination Data Analysis

This project explores global COVID-19 vaccination trends using data from **Our World in Data**. It includes data preprocessing, exploratory data analysis (EDA), and visualizations using Python libraries like **Pandas**, **Seaborn**, and **Matplotlib**.

## 📁 Files
- `Vaccination_data.ipynb` – Main notebook
- `country_vaccinations.csv`
- `country_vaccinations_by_manufacturer.csv`

## 📊 Highlights
- Top vaccinated countries
- Daily trends visualization
- Vaccine manufacturer comparison
- Heatmaps and outlier detection

## 🛠 Tools
- Python, Pandas, Matplotlib, Seaborn
- Google Colab

## 📚 Data Source
- [Our World in Data – Vaccinations](https://ourworldindata.org/covid-vaccinations)

## ✅ Summary
We analyze global vaccination rollout, visualize trends, and draw insights into vaccine usage across countries using open datasets and Python.
